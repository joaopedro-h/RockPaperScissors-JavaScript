🎮 Pedra, Papel e Tesoura.

Um minigame de Pedra, Papel e Tesoura desenvolvido em JavaScript utilizando Node.js e executado diretamente no terminal (CLI).

O jogo permite que o usuário jogue contra a máquina, que escolhe sua jogada aleatoriamente.

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de arrays, geração de números aleatórios e interação com o terminal.


🚀 Tecnologias utilizadas

JavaScript

Node.js

Módulo nativo readline

Terminal / CLI
